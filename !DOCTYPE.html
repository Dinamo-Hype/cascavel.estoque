Sobre o Hyper Gerenciador

O Hyper Gerenciador é um sistema completo de gerenciamento de estoque desenvolvido para pequenas e médias empresas, oferecendo todas as ferramentas necessárias para um controle eficiente de produtos e fornecedores.

Perguntas Frequentes

Para adicionar um novo produto, navegue até a seção "Produtos" no menu lateral e clique no botão "Novo Produto". Preencha todos os campos obrigatórios no formulário e clique em "Salvar". Certifique-se de incluir informações como nome, código, preço e quantidade em estoque.

Na seção "Relatórios", clique no card "Relatório de Estoque". O sistema irá gerar automaticamente um relatório com todos os produtos em estoque, incluindo quantidades e valores totais. Você pode exportar este relatório para PDF, CSV ou JSON usando os botões na parte superior da tela.

Vá para a seção "Configurações" e depois para "Backup e Restauração". Clique no botão "Criar Backup" para baixar um arquivo JSON com todos os dados do sistema. Recomendamos fazer backups regularmente para evitar perda de informações.

Na seção "Configurações", no campo "Limite de Baixo Estoque", defina a quantidade mínima que um produto deve ter para não ser considerado em baixo estoque. O sistema irá alertar você quando algum produto atingir essa quantidade.

Sim, todos os dados ficam armazenados localmente no seu navegador. Nenhuma informação é enviada para servidores externos. Para maior segurança, recomendamos fazer backups regulares e proteger seu computador com senha.
